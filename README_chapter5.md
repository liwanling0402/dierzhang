# 第五章：子图绘制 - 交互式可视化平台

## 项目介绍

本项目基于《Python数据可视化》第五章内容，构建了一个功能丰富的交互式数据可视化平台。平台展示了多种子图绘制技术和数据可视化方法，提供了直观的用户界面和丰富的交互功能。

## 功能特点

### 📊 核心功能模块

1. **产品销售分析**
   - 产品A与产品B的销售额月度对比
   - 销售数据饼图分布展示
   - 多维度数据分析

2. **宠物养宠比例分析**
   - 各国养猫人群比例横向条形图
   - 各国养狗人群比例横向条形图
   - 国际养宠习惯对比分析

3. **抖音用户分析**
   - 2017-2018年抖音用户增长倍数对比
   - 用户地区分布饼图展示
   - 社交媒体用户趋势分析

4. **气象数据分析**
   - 平均气温与降水量、蒸发量的关系
   - 混合图表展示（柱状图+折线图）
   - 环境数据综合可视化

5. **汽车销售分析**
   - 2018年上半年汽车销售额月度分析
   - 分公司销量对比
   - 销售数据面积图展示

### 🎨 界面特性

- **响应式设计** - 适配各种屏幕尺寸
- **玻璃拟态效果** - 现代化的视觉设计
- **渐变背景** - 美观的视觉效果
- **平滑动画** - 流畅的用户体验
- **交互式控制面板** - 丰富的自定义选项

### ⚙️ 高级功能

#### 布局控制
- 网格布局模式
- 流式布局模式
- 自定义布局模式
- 子图数量调整（1-9个）
- 响应式断点设置

#### 数据控制
- 数据筛选（全部/近期/热门/自定义）
- 自动刷新功能（手动/5秒/10秒/30秒）
- 数据聚合方式（求和/平均/最大值/最小值）

#### 视觉效果
- 多种动画效果（无/渐变/滑动/弹跳）
- 配色方案切换（默认/柔和/鲜艳/单色/自定义）
- 交互模式设置（悬停/点击/拖拽/缩放）

#### 导出功能
- 多种导出格式（PNG/JPG/SVG/PDF）
- 分辨率设置（标准/高清/超高清）
- 导出选项自定义

## 技术栈

- **前端框架**: HTML5 + CSS3 + JavaScript
- **UI框架**: Tailwind CSS
- **图表库**: Chart.js
- **图标库**: Font Awesome
- **动画**: CSS3 Animations + JavaScript

## 使用方法

### 基本使用

1. 直接在浏览器中打开HTML文件
2. 选择不同的功能模块进行查看
3. 使用控制面板调整显示参数
4. 点击图表进行交互操作

### 高级功能

1. **切换模块**: 点击顶部导航按钮切换不同的数据模块
2. **调整布局**: 使用控制面板选择不同的布局模式
3. **筛选数据**: 通过数据筛选功能查看特定范围的数据
4. **导出图表**: 使用导出功能保存可视化结果

## 交互说明

### 导航栏
- 点击模块按钮切换不同的数据可视化内容
- 使用主题按钮切换界面主题（开发中）
- 点击统计查看数据统计信息（开发中）
- 使用导出按钮导出当前视图（开发中）

### 控制面板
- **布局控制**: 选择图表的排列方式
- **数据控制**: 筛选和聚合数据
- **视觉效果**: 调整动画和配色
- **导出设置**: 配置导出参数

### 图表交互
- 鼠标悬停查看详细数据
- 点击图例显示/隐藏数据系列
- 支持缩放和平移操作

## 数据来源

本平台使用的数据来源于《Python数据可视化》第五章的示例数据：

- **产品销售数据**: 模拟的月度销售额数据
- **宠物养宠数据**: 各国家养猫养狗人群比例统计
- **抖音用户数据**: 2017-2018年用户增长和地区分布数据
- **气象数据**: 月度气温、降水量、蒸发量数据
- **汽车销售数据**: 2018年上半年汽车销售统计
