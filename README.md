# 数据可视化交互式平台 - 第2、3、4章集成

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://liwanling0402.github.io/datavis24/)
[![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

一个基于matplotlib的完整交互式学习平台，集成第2、3、4章内容，全面展示数据可视化技术的艺术之美。

## 📊 项目简介

本平台是数据可视化教材第2、3、4章的完整集成实现，通过交互式界面让用户深入学习和体验matplotlib的各种图表绘制和样式优化技术。

## 🌟 核心功能

### 📊 第2章：基础图表绘制
- **折线图应用** - 数据变化趋势可视化展示
- **柱形图技术** - 分类数据比较和统计
- **气温走势分析** - 双线图实时数据展示
- **电影票房排行** - 柱形图数据可视化

### 📈 第3章：多种图表类型
- **三角函数演示** - 正弦余弦函数图形展示
- **箱形图分析** - 数据分布统计分析
- **散点分布图** - 多变量关系探索
- **实时动态图表** - 交互式数据生成

### 🎨 第4章：样式优化技术
- **颜色应用** - 基础颜色、渐变效果、颜色映射
- **线型选择** - 线条样式、标记符号、线宽配置
- **字体设置** - 中文字体支持、文本样式优化
- **主题风格** - 多种主题切换、实时预览
- **区域填充** - 曲线间填充、透明度控制
