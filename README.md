# 第4章：图表样式优化交互式平台

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://liwanling0402.github.io/dierzhang/)
[![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

一个基于matplotlib图表样式优化的完整交互式学习平台，专注于数据可视化的艺术之美。

## 📊 项目简介

本平台是数据可视化第4章"图表样式优化"的完整实现，通过交互式界面让用户深入学习和体验matplotlib的各种样式优化技术。

## 🌟 核心功能

### 🎨 颜色应用
- **基础颜色配置** - 使用十六进制、RGB等颜色格式
- **渐变颜色效果** - 线性渐变、径向渐变等
- **颜色映射表应用** - 使用matplotlib内置颜色映射
- **实例展示**：图书采购情况堆叠柱形图

### 📈 线型选择
- **线条样式** - 实线、虚线、点线、点划线等
- **标记符号** - 圆形、方形、三角形、星形等
- **线宽配置** - 不同粗细的线条效果
- **实例展示**：美元/人民币汇率走势对比

### 🔤 字体设置
- **中文字体支持** - 黑体、雅黑、楷体等
- **字体大小动态调整** - 标题、标签、文本等
- **文本样式** - 粗体、斜体、下划线等
- **实例展示**：气温走势图数值标签

### 🎭 主题风格
- **多种主题切换** - 默认、浅色、深色、学术等
- **实时预览** - 切换主题立即看到效果
- **风格对比** - 不同主题风格对比展示

### 🎯 区域填充
- **曲线间区域填充** - 多边形和曲线间填充
- **透明度控制** - 不同透明度效果的填充
- **复杂图形填充** - 科赫雪花等复杂图形

## 🎮 交互功能

### 平台操作
- **响应式设计** - 适配桌面和移动设备
- **深色/浅色主题** - 右上角🌙按钮切换
- **键盘快捷键** - Ctrl+1到6快速切换标签页

### 实时控制
- **参数调节** - 实时调整各项样式参数
- **效果预览** - 立即看到参数变化效果
- **主题切换** - 多种主题风格实时切换

## 📚 学习内容

### 第4章核心知识点
1. **图表样式基础** - 默认样式和自定义样式
2. **颜色应用技术** - 颜色选择和配置方法
3. **线型标记设置** - 线条样式和标记符号
4. **字体文本优化** - 字体选择和文本样式
5. **主题风格定制** - 主题切换和风格定制
6. **区域填充技术** - 多边形和曲线填充

### 实际案例
