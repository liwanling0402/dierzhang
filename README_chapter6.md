# 第六章：坐标轴 - 交互式可视化平台

## 📚 章节概述

第六章深入探讨了matplotlib坐标轴的高级定制技术，包括坐标轴结构、任意位置创建、刻度设置、轴脊控制等核心知识点。本章通过丰富的交互式实例，帮助学习者掌握坐标轴的精细控制方法。

## 🎯 核心功能模块

### 1. 📊 坐标轴概述
- **学习目标**：理解坐标轴的基本结构和组成
- **主要内容**：
  - 轴脊（Spines）：坐标轴的四条边线
  - 刻度（Ticks）：轴上的标记线
  - 刻度标签（Tick Labels）：刻度的文字说明
  - 轴标签（Axis Labels）：轴的名称
  - 网格线（Grid）：辅助参考线

### 2. 📍 任意位置添加坐标轴
- **学习目标**：掌握在画布任意位置创建坐标轴的方法
- **核心技术**：
  - `add_axes()`: 精确位置控制
  - `subplot2grid()`: 网格布局
  - `gridspec`: 复杂网格布局
- **坐标系统**：[left, bottom, width, height] 精确定位

### 3. 🌊 深圳风速分析
- **学习目标**：时间序列数据可视化实践
- **数据特征**：
  - 最低风速：2.5 m/s (0:00)
  - 最高风速：6.9 m/s (14:00)
  - 平均风速：4.6 m/s
  - 变化趋势：早晚低，午后高
- **可视化技术**：折线图 + 平均线组合

### 4. 📈 正弦余弦曲线
- **学习目标**：三角函数图像绘制与分析
- **函数特征**：
  - 周期性：周期为 2π
  - 振幅：范围为 [-1, 1]
  - 相位差：cos(x) = sin(x + π/2)
  - 对称性：sin为奇函数，cos为偶函数

### 5. 🎛️ 轴脊控制
- **学习目标**：轴脊定制技术
- **控制方法**：
  - 隐藏轴脊：`set_visible(False)`
  - 移动轴脊：`set_position()`
  - 设置颜色：`set_color()`
  - 设置线宽：`set_linewidth()`
