# 交互式数据可视化平台

## Core Features

- 多格式数据导入

- 动态过滤排序

- 多种图表类型

- 交互式操作

- 响应式布局

- 图表导出功能

## Tech Stack

  "Web": {

  "Backend": "Node.js + Express.js",
  "Visualization": "D3.js + Recharts"

## Design

现代Glassmorphism风格，深蓝色科技主题，半透明毛玻璃效果，流畅动画交互
